# Distribution-Feeder-Optimizer
My project is a three phase distribution system, I am given the base voltage, the power consumed for the three loads, and also the respecitve power factor. I calculate the line currents through a recursive method I initally assume that the node connected to the load has the voltage of the substation voltage. I then calculate the current going into the load by getting getting the conjugate of the apparent power, and divviding it by the voltage of the node. The line currents are then calculated :add the associated load currrent to the line current in front of the one being calculated, and this process is repeated recursively until all line currents are found. Through, the use of line currents, the substation voltage, and the assumed impedances per mile, I calculate the node voltages. I repeat this process until the voltage values converge. Startign with calculating load currents again.
